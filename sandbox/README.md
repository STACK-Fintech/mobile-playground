# Setup
* sandbox-api
  1. `cd sandbox`.
  2. `npm install`.
  3. `npm start`.

* ngrok
  1. Sign up a free account at https://ngrok.com/.
  2. Download `ngrok` and `unzip /path/to/ngrok.zip`.
  3. `./ngrok authtoken [TOKEN]` to authenticate. 
  4. `./ngrok http [PORT_NUMBER]` (`PORT_NUMBER` is `3000`).
