# Setup
* Android: Android Studio (3.6 or higher) with gradle and emulator (API 23+).
  1. `cd android`.
  2. Import project in Android Studio.
  3. Run on emulator.
  
* iOS: XCode 12+
  1. `cd ios/app`.
  2. pod install //cocapods installation guide: https://guides.cocoapods.org/using/getting-started.html
  3. Run on simulator.
